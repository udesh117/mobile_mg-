## ğŸ’¡ Key Points

### âœ… What You Need to Know:

1. **`userId` is generated by Firebase Auth**
   - Created automatically when user signs up
   - Unique for each user
   - Format: Long alphanumeric string

2. **Project owner is automatically a member**
   - When you create a project, you're added as a member
   - You can assign tasks to yourself

3. **Assignee = Project Member's userId**
   - When you select an assignee, you're selecting a project member
   - Their `userId` is stored as `assigneeId` in the task

4. **Currently only project owner can be assigned**
   - No UI to add other members (not required)
   - But the system supports it if you add members later

## â“ Common Questions

### Q: Where does the userId come from?
**A:** Firebase Authentication generates it automatically when user signs up. You don't create it manually.

### Q: Can I see the full userId?
**A:** Currently, only first 8 characters are shown in UI. Full userId is stored in database.

### Q: Can I assign tasks to other people?
**A:** Currently only to project owner (yourself). To assign to others, you'd need to add them as project members first (no UI for this yet, but hooks exist).

### Q: What if I don't assign a task?
**A:** That's fine! Tasks can be unassigned. Just leave assignee as "Unassigned".

### Q: How do I change assignee?
**A:** Tap on task â†’ Edit modal â†’ Change assignee dropdown â†’ Save
